# Features

## Posts (Blog/Memo)
- ブログ/メモ投稿
- タグ付けとタグ検索
- いいね/コメント
- アルバムとN:Nで紐づけ可能
- アルバムに紐づかない投稿も作成可能

## Albums/Photos
- アルバム作成
- 写真アップロード（S3署名URL）
- 写真と投稿を関連付け可能

## Invitations
- 招待制（管理者がメール送信）
- 管理者は環境変数で複数指定

## Notifications
- Web Push通知（FCM）
- 通知カテゴリ: new_post, new_comment, anniversary, trip
- 投稿/コメントは即時通知
- 記念日/旅行は登録時に通知タイミングを指定
- カテゴリごとのON/OFF設定

## Anniversaries
- 記念日を登録
- 通知タイミング指定

## Trips
- 旅行を登録
- 旅程（時間単位）
- 行きたい場所リスト
- 旅費の予算/実績（項目ごと）
- 通知タイミング指定
